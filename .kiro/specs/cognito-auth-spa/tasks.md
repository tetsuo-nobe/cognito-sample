# 実装計画: Cognito認証SPA

## 概要

Amazon Cognito User Poolを使用した認証機能付きSPAの実装計画。認証方式はユーザー名（Username）とパスワードによるサインインで、Eメールは必須属性として登録時に収集する。フロントエンドはVanilla HTML/CSS/JavaScript、バックエンドはAPI Gateway REST API + Lambda（Python 3.13）で構成する。ハッシュベースのSPAルーティングにより、サーバー設定不要の静的ホスティングに対応する。

- タスクに `*` が付いたものはオプションであり、MVP実装時にはスキップ可能です
